Первый запуск
Запустите AntivirusApp
При первом запуске программа откроется в режиме ожидания
Для полноценной работы необходимо подключить VirusTotal API Key
5. Получение VirusTotal API Key (ОБЯЗАТЕЛЬНО)

Без API-ключа облачное сканирование работать НЕ БУДЕТ.

Шаг 1. Регистрация на VirusTotal
Перейдите на сайт:
https://www.virustotal.com
Нажмите Sign Up
Зарегистрируйтесь:
через Email
или через Google-аккаунт
Подтвердите регистрацию (если потребуется)
Шаг 2. Получение API Key
После входа в аккаунт нажмите на иконку профиля (в правом верхнем углу)
Перейдите в раздел:

API key
Скопируйте ваш Personal API Key
(выглядит как длинная строка из букв и цифр)
⚠️ Никому не передавайте ваш API-ключ
6. Подключение API Key в программе
Откройте AntivirusApp
Перейдите во вкладку:
Settings (Настройки)
Найдите раздел:
VirusTotal Settings
В поле API Key:
вставьте скопированный ключ
Нажмите:
Save Settings
Перезапуск программы не требуется
7. Сканирование файлов
Перейдите на вкладку Home
Нажмите:
Select File for Scanning
Выберите любой файл на компьютере
Дождитесь результата проверки
Возможные результаты:
✅ Угроз не обнаружено
⚠️ Threat Detected (обнаружены угрозы)
8. Действия при обнаружении угрозы
Если файл признан опасным:
программа уведомит пользователя
станет доступно:
Удаление файла
Перемещение в карантин (если включено)
Настройки поведения можно изменить в разделе Settings
9. Карантин и логи
Карантин хранится в папке:
Program Files\AntivirusApp\quarantine
Журналы событий:
Program Files\AntivirusApp\logs
Автоматическую очистку логов можно включить в настройках.
10. Возможные проблемы
🔸 Сообщение «No API Key found»
→ Укажите API-ключ VirusTotal в настройках
🔸 Предупреждение Windows Defender
→ Нажмите Подробнее → Всё равно выполнить
(нормально для пользовательских приложений)
